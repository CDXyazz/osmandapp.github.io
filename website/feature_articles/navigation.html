The navigation function allows you to reach your destination easily using voice guidance (optional).

To start the navigation, you need to use the navigation button on the map screen. Next, you'll need to set the starting point and the destination. As for the starting point, you can choose your current location, Favorite location, select a point on the map or use an address. You can also set a starting point by long tapping on the map and choosing 'Directions from' in the menu box. To navigate to a point, just press the navigation button in its context menu. As soon as your points are selected, the app will create a route and will start guiding you after you tap Start.

When the navigation is running, you can press the home button. You will get the message 'OsmAnd is running in the background'. This means that even if you turn the screen off or exit the active app window, you'll still get voice prompts. For Android versions other than 3.3, we have added the “Turn on screen” option. This allows you to show the map on the lock screen during navigation. Now, this function does not request any permissions for the correct operation. You can use it to save phone power. The function is configured separately for each profile. To configure, you need to select a profile that supports navigation and go to Profile settings - Navigation settings - Map during navigation - Turn on the screen (move the slider to the ON state - should turn blue).

Navigation profiles (for Android version only)

In OsmAnd, to build a route, various basic profiles are available that take into account the specifics of the vehicle and traffic conditions. Routes are calculated differently for different navigation profiles following the rules of the road, given conditions, weights, various restrictions and will help you get to your destination, taking into account your method of movement. We also believe that it is not very convenient to use basic profiles for some special cases. Therefore, OsmAnd 3.5 added the ability to make your own profile with all the settings, icons, etc.

At the top of the main menu, you can see the active navigation profile and proceed to its configuration. You can set different parameters and special plugins for each profile separately.

To create a new profile based on the basic one, select in the general menu "Settings" - "+ New Profile ". In the same settings, you can import a profile from a file. To get the profile file for import, go to "Settings" - select the profile you want to export - "Export profile". In the same menu you can reset to default values.

You can read more about creating and configuring a navigation profile in this article.

Voice guidance

Android

You can receive voice prompts while driving. To receive them in your language, select a profile with the ability to navigate in the main menu, click "Configure profile", then go to "Navigation settings" - "Voice prompts" - "Voice guidance" and select your preferred language or click "Intstall more" to set your preferred. If you want your music to pause when scoring a route, in the same menu (Language and Output), turn on the “Pause music” option.

There are many settings for voice prompts on this screen. You can choose any variant for you.

You can turn off voice prompts if you don’t need them right now. To turn off voice prompts, go to the main menu "Configure profile"-> "Navigation settings"-> "Voice prompts". You can also turn on or turn off voice prompts on the icon of sound on the navigation screen.

What are TTS and pre-recorded voice prompts

OsmAnd has two types of voice guidance: TTS or simpler text-to-speech, and recording created from pre-recorded voices. You can switch between them to choose the type that is most convenient for you. TTS is preferable as it functions more flexible. Select the type of voice prompt you like in the main menu "Configure profile"-> "Navigation settings"-> "Voice prompts"-> "Voice guidance".

Please read more about setting up voice navigation here: Setting voice navigation.

iOS

To set the language for your voice prompts, please go to app menu-> Settings-> Navigation settings-> choose "Car" (for example)-> Voice guidance and select voice guidance for navigation.

There is only TTS or simpler text-to-speech for iOS.

You can also turn on or turn off voice prompts on the icon of sound on the navigation screen.

Navigation services

Android

In the OsmAnd navigation settings for various profiles, you can select the type of navigation. For a type of navigation, you can select some basic navigation profiles as well as the Straight line.

The Straight line option connects your waypoints with a line: it can be used for general trip planning when you do not need a detailed route. Other services have their own algorithms of route calculation, you can test them to see which one meets your needs.

Navigation settings

You can read information about navigation settings here for Navigation profiles.

We can choose any options for your route navigation after building a trip. Click to Options on Navigation screen.

Remember, that for each Navigation type these settings are different. You can improve your route using certain route parameters. Look at screens below for comparing it.

In Navigation type Bicycle you can set Driving style: 1.Unpaved roads 2.Byways 3. Balanced 4. Shorter routes for choosing your prefering style, you can use Elevation data for consideration of Ascent / Descent for your trip, use or not Motorways.

In Navigation type On foot you can can use Elevation data for consideration of Ascent / Descent for your trip.

In Navigation type Car you can use Fuel-efficient way and Allow private accessfor your trip.

In Navigation type Boat you can use all standart settings.

In Navigation type Third-party you can choose Offline calculation of OsmAnd route segment.

How to avoid certain roads

You can customize your route to avoid certain streets or types of roads. To do this, create a route, then enter "Options button"-> "Avoid roads..." in the navigation screen. You can avoid toll roads, unpaved roads, border crossings, ice roads or fords, ferries, shuttle train, motorways. Choose the options you need. You can also avoid a certain street. Maybe some construction work is underway, or there is a traffic jam or something else that makes you avoid this stretch of road. Just click “Select on map” in "Avoid roads" screen for changing your route.

How to add/remove waypoints

To add or remove waypoints, you need to make a long press on the map in the place you want to visit, do it after the route is calculated, click “Route”> “Add as last intermediate destination” or other menu items as you wish. Add several waypoints one by one, if necessary, in the menu of navigation details by pressing the + button which is located to the right of the label with the destination. When you add a new destination, the previous one can be edited by clicking the pencil icon - "Edit" . In the menu that appears, you can drag destination points with (≡) button, changing the order of their visits along the route, as well as delete route points by pressing 'X' or add new ones.

How to pause, resume or stop navigation

Need to pause your navigation to make a quick stop? That's easy. To pause, resume or stop your navigation completely, go to your dashboard and press the button on your navigation menu next to the Destination info. To stop it, tap on the screen during navigation and pressing the 'X' icon in the context menu.

How to view POI along the road

You can choose to view points of interest along your route. To do this, create a new route, then click the gear icon "Settings" in the navigation context menu, then click Show along route . Select Points of Interest (POIs). In the menu that appears, select which Places of Interest you want to see along the route. After that, specify the maximum distance from you to the points of interest that will be displayed. Use the 'X' buttons to remove POIs that you do not need.

How to save navigation route as GPX file (only for Android version)

You can build a route and save it as a GPX file for later use. Thus, you do not have to wait for the calculation of the route, and you will have a reliable source of navigation. To save the current navigation route as a GPX file, go to the navigation context menu and click Details. Click the “Download” button in the upper right corner and name the GPX track to save it. To use it for navigation, go to the main menu - “Map Settings” - “GPX Files” - select your track and click "OK" . You will see the track on the map. Then press the navigation button. There, in the "Displayed tracks" menu, select the desired route and then click "Start" to start navigation.
