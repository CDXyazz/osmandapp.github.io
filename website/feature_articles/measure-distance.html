Measure distance

  • How to use it
  • Create or edit GPX track

The 'Measure distance' allows you to measure distances on the map, create and edit GPX tracks.

To enable this feature, simply go to the main OsmAnd menu and press 'Measure distance'. You'll see the map with a tool bar below. To measure the distance, start by butting the first point to the map and adding the next ones (one or more). You'll see the total distance on the toolbar. You can also undo or redo the steps using the arrow buttons.

How to use it

After you've done adding points, press 'Options'. There, you can try several convenient features:

  • Snap to road - this function creates a route using all your points. You can set the car, bicycle or pedestrian mode, as usual
  • Save as new GPX track - allows you to save your editing as a set of points or a line
  • Add to a GPX track - adds the line you've created (in a fort of straight segments) to a GPX track of your choice
  • Clear all - removes all the points.

You can also measure a distance using radius ruler tool.
