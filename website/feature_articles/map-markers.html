What are map markers

Markers interface is a convenient tool that helps you save points on the map for your short-term needs. While favorites are often places you'll visit multiple times (like your gym, friend's house, etc.), markers are used to locate the places you may need once: a shop you'll need to stop by, a gas station on your long trip, etc.

Map markers for Android

Main features

To use the markers, just make a long tap on the map and select the flag icon in the context menu. Done. Now, this point is your map marker. All of them are stored in the "Map markers" menu in OsmAnd.

You can view the list in the "Map markers" menu. This interface is extremely flexible, so there are many ways you can arrange the markers. In the "Map markers" menu, you'll see three main tabs:

List This tab shows the list of markers and allows you to change their order. To rearrange your markers, simply use the '≡' button to drag them up or down the list. To remove a marker, press the '✓' button next to it.

Groups This tab sorts the markers by date which allows you to conveniently pick those created on a certain day. For example, you can view all the points you added the day before. You can import your favorites as a separate markers group and do the same with your GPX waypoints. Simply go to Map markers-> Groups and press '+'.

History Here, you'll see the markers you removed. To restore any of them, use the '↻' button next to it.

Advanced options

There are some advanced options you can try. Press the "Options"(More) menu to try the following:

Sorting your markers

Select the "Sort by" option to sort the points by the distance, by name or by the date they've been added (ascending or descending).

Appearance on the map

You can set how you'd like to view your markers - using the top bar on the map screen or the widgets. You can specify number of direction indicators: one or two. Or you can switch off Distance indication.

This menu also allows you to view the guide lines to the markers or show their direction using arrows. To try it, enable "Show arrows on the map" and "Show guide line" features.

More, you can choose the option "One tap active", when tapping a marker on the map to move it to the top of the active markers without opening the context menu.

The option "Keep passed markers on the map" allows to add markers as a group of Favorites or GPX waypoints marked Passed will remain on the map. If the group is not active, markers will disappear from the map.

Adding markers by coordinates

Besides selecting a point on the map, you can also enter its coordinates. To try it, please go to Map markers-> Options-> Coordinate input.

Enter the coordinates, then name your marker. Press "Options" for additional features: "Use system keyboard" sets your device's keyboard as default, "Go to next field" automatically switches to the next field when the current one is completed. "Digits quantity" selects how many digits will your coordinate indicator have.

You can save your markers here like GPX track: Options-> Save as track.

Plan route

Planning a route from your map markers is very easy. Press "Plan route" and select the markers you need. You can also customize the route. After you've selected the markers, press "Options" in the upper-right corner and chose the additional features you need.

Press "Navigate" to create a navigation route using the points. If you need to return to your first point, select "Make a round trip". "Sort door-to-door" option creates a route by arranging the points in the most convenient way (based on their distance to each other). The "reverse order" function reverses the last selected order of the points. (So if you select door-to-door and then reverse, the points will still be sorted door-to-door, but the route will be created in the opposite direction.)

Saving the markers as a GPX track

To conveniently save and share all your points, use Map markers-> Options-> 'Save as track'. Name it and save.

Move all markers to history

To move all markers to history at once, select 'Move all to history'. This help if you have a long list of points.
