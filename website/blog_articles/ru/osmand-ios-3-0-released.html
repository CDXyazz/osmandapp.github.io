OsmAnd 3.0 (iOS)

September 03, 2019

We're glad to announce the new OsmAnd version for iOS.

• Added Quick Action feature: instant access to various map actions (e. g. changing a map style, showing specific POI types etc.) directly from the main screen

• Context menu improvements: ability to download the missing region map directly from the context menu

• OpenStreetMap editing: added hints for tags and values to the advanced editing screen

• Improved the trip recording widget UI

• Fixed an issue with navigation alerts not being refreshed

• Various improvements and bugfixes

And remember that only together we can achieve the best results!
