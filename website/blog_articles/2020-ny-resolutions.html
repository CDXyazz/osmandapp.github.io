2020 New Year resolutions

Dec 31, 2019

Happy New 2020 Year!

Every year we follow tradition to write New Year Resolutions. First of all we would like to describe our achievements, compare them with 2019 Resolutions and prepare a new list for 2020.

2019 achievements

First of all, we would like to congratulate our CEO Victor Shcherb who was awarded during SotM-2019 in Heilderberg, Germany (Ulf Möller Memorial Award) for creating OsmAnd application and supporting it over the years, keeping it the only major OSM-based app with no corporate influence, and constantly imagining ways to expand user-facing side of open data.

Android

OsmAnd Android got 3 major releases this year (3.3, 3.4, 3.5) with more than 300 public features and issues closed on Github. Among all these features we’ve selected 3 major features which were in the Resolutions list for 2019.

  • Public transport

    We’ve introduced Navigation for public transport in early version 3.3 and did multiple fixes in 3.4 as well. You can also watch the presentation on Youtube.

    It was an important step to introduce Public Transport worldwide without limits. We dealt with thousands map issues and we welcomed OSM-community to improve the map and use OsmAnd as a test application.

    We know that many users are waiting for schedule support or at least [opening_hours] + [interval]. And we have plans to improve it in 2020.

  • Application profiles

    New application profiles is our long term goal and we’ve started working on it this year. There is a lot of work left for next 2020 though we try to bring something new each release to give an impression of the end goal and get a necessary feedback from our users. So far all settings are made separate, so you can customize application in various ways for different purposes.

    The most important update was to allow to create custom profiles and have button to import and export settings associated with this profile. You can choose a specific icon and color to have a unique look and feel. Next year we are expecting to include associated Quick Actions, Maps, custom Poi Categories, Plugins etc.

  • Powerful Route details

    As of 3.3 route details screen have important information about road types, surface conditions, steepness displayed in a nice visual way. It is a top priority for us to migrate that functionality to iOS. This mechanism quite advanced and everybody could contribute and create own metrics by modifying just rendering style on own device. Thanks to Peter who contributed road information for Ski Pistes this way.

  • What else?

    We’ve redesigned Directions screens, improved Quick action, added a Widget with Current Coordinates and the Compass to the Radius ruler, added Ski Routing, updated Basemap, Desert style for map and lot’s of else.

    iOS

    We did 7 releases of OsmAnd for iOS this year (2.5, 2.7, 2.8, 2.9, 3.0, 3.10, 3.11)! All this was possible because of high commitment from our iOS team to catch up with Android. We plan that 2020 will be a special year when almost all features will be equal between Android and iOS. Here the list of main features released this year for iPhones and iPads.

  • OsmAnd Live

    OsmAnd Live is a tiny subscription that provides you numerous benefits, such as unlimited downloads, access to all paid features and map hourly updates.

  • OpenStreetMap (OSM) Editing

    OSM Editing allows you to make contributions to OpenStreetMap, a global community aimed at creating a comprehensive map of the world and providing up-to-date open-source data to every user.

  • Mapillary for iOS

    We're proud to have collaboration with Mapillary available on iOS as well. Now you can view the street-level imagery in your location or anywhere in the world using the Mapillary.

  • Quick Action menu

    This feature was asked many times this year and we’re proud to deliver it in 2019. It allows you to have instant access to various actions with 2 clicks from the main screen, e.g. to change a map style, to show specific POI, to switch between raster maps or disable some layers.

  • Support of Online SQL maps

    First of all, we’ve added more maps to "Online maps". Secondly, we provided feature to support the same map sources as we have on Android including *.sqlite with dynamic online loading.

  • New Navigation Screen and Graphs

    Now you can add "Home", "Work" point of destination. The short way to Options menu. When your trip is ready, you can see the altitude/slope graph.

    OsmAnd Online GPS Tracker

    During this year we made 3 releases of our Tracker (0.4, 0.5, 0.7).

    New Year Resolutions 2020

    New year we are going to start with an extended team of 7 software engineers, we grew to 20% compared to the previous year and it is good news, so we can probably do 10% more in 2020 :-)

    Our list is growing every year and as of today, we have around 1500 requests waiting to be fixed. That’s why it is important for us to set up some public goals and try to follow it to a certain extent.

    GENERAL

    • New slopes / hillshade maps
      In 2019 we’ve released updated contour lines worldwide and started working on new raster hillshade and slope maps. We want to provide more detailed raster maps and also add a new type of maps to better visualize slope.

    • Profiles on Android + iOS
      This is considered to be the most important feature for 2020. It was already started in 2019 and now Android users can create / import and export custom profiles with user-defined settings. We’re going to continue profile’s customization, so every profile could look and feel as a separate geo-specialized application. Also we want to make it easy to share with others.

    Android

    • OpenGL-rendering + 2.5D
      This is a long running feature for several years and we have a prototype build, we’ve already added some features to our roadmap to be implemented. Though we are not planning to make it mainstream unless it will be very successful. You can always check out the version we have at our build server.

    • Travel + Wikivoyage improvements
      This year we didn’t manage to implement many features related to Wikivoyage, though it is always in our focus. We plan to make a second version inspired by our new plugin architecture.

    • Local GPX-access
      Every year we develop many tiny and useful features related to GPX and this 2019 year was not an exception. We’ve closed more than 50 GPX-related issues. In 2020 we plan to add new features how to interact with GPX on the map, how to browse and search among large list of GPX files and new ways how to visualize GPX-activities.

    iOS

    • Public Transport
      This feature was done for Android in 2019 and we plan to migrate all solutions we implemented to iOS in 2020. So OSM-ers with IPhones will get a new motivation to map public transport in OSM (check out our guideline).

    • Measure Distance
      We’ve got a lot of requests that this tool is very helpful and for sure it is number 1 priority to get it in IOS in 2020.

    • GPX
      As Android version gets more advanced-GPX related features, iOS version still lacks some base route graphs and stats. Of course it is very important for us not to miss it next year.

    Sure, many of hundreds of small features didn’t get to this list though they’re going to be developed in 2020.

    Happy 2020 and let’s welcome OsmAnd 2020!!!
