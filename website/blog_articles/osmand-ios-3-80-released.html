OsmAnd 3.80 (iOS)

September 10, 2020

For our team a release day, it's like a progress report to the users.

Every day we are working on the app improvements to make it more user friendly. We fix mistakes, that also occur due to massive changes that turned basic maps into a universal tool for travellers for over 10 years.

But still, it is not perfect yet. We are not only grateful to all the positive reviews, but also thankful for the constructive criticism and your dream of a better OsmAnd.

Furthermore, here is a list of this Summer updates:

  • Introduced “Application Profiles" with independent settings
  • New arrangement of settings by profiles and types: navigation, general, map and screen
  • Ability to import or export profile settings
  • Added data sources for Online photos
  • Added ability to download online maps to cache
  • Improved "Search" algorithms
  • What else is in this release?

    • Introduced “Application Profiles" with independent settings

    At this release we added Application Profiles. Now you can create your own profile with independent settings.

    Go to menu -> Settings -> New profile. Here you can change an order in profile list, switch on/off profiles.

    How to create new profile you can read here.

    New arrangement of settings by profiles and types: navigation, general, map and screen

    Now each profile has own settings. Go to menu -> Settings -> сhoose profile.

    Each profile has Genearal settings, Navigation settings, Configure map, Configure screen and Profile appearance menu.

    Of course, you can set Plugins for selected profile. About settings and navigatio profile read here.

    Ability to import or export profile settings

    You can export your profile or import other profile.

    To import a profile, tap on the needed file (*.osf) in any application and select "Copy to OsmAnd Maps".

    To export a profile, go to menu -> Settings -> Choose profile -> Export profile.

    Added data sources for Online photos

    We added more data sources for Online photos except mapillary-photos: wikidata, wikimedia commons. When you click to POI or polygon you find more pictures in Online photos.

    Added ability to download online maps to cache

    Now you can download online maps to cash.

    At first, you should change online maps in 'Map type' menu. Next, long click to the map -> Actions -> Download map. Here, you can choose map area, map type, max/min zoom, you see number of tiles and download size. When you set all parameters for downloading to click 'Download'.

    Improved "Search" algorithms

    We fixed many bugs in our search algorithms. Now you can find places, points and etc. without any problems.

    What else is in this release?

    • Fixed crash while starting navigation

    • Fixed an issue with "Nautical maps" always stuck in update state
