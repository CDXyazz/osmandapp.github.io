OsmAnd 2.6

Apr. 5, 2017

We're glad to announce that version 2.6 has been released! We're proud to present some major improvements and new features and hope you'll enjoy them. So, let's come down to the details.

Quick action button

Finding the balance between adding all the necessary features and yet keeping the user interface simply has always been a challenge for us. We were looking for the ways to preserve all the features that come in handy for tourists, mappers, cyclists, drivers and travelers. At the same time, it was necessary to make OsmAnd convenient for day-to-day use. In the version 2.6, we decided to add the quick action button. It is a menu providing access to your most frequently used features. You select and add them to this 'toolbar', so the next time you need to add a new favorite, add navigation destination or enable voice guidance, you can do it in seconds. The quick action button is available in Configure screen menu. Please see full video guide below:

Improved search menu

Some users reported they liked the new simple search field, while others said they preferred the previous step-by-step search better. So we decided to combine both approaches. Now, if you open the Search menu, you'll see three tabs: History, Categories, and Address. The third one allows you to search by address, by postcode or by coordinates, like it was in the previous versions.

Elevation and slope profile for recorded tracks and calculated routes

This feature has been long requested, especially by hikers and tourism enthusiasts. Now you can see very detailed info about your navigation route or GPX file. Just calculate a route, then tap the information button in the navigation context menu (right below the ...). You'll see the route's elevation profile. To view it along with the route on the map, please select 'Analyze on map' menu. More to come, now you can view it in Altitude, Slope mode or combine them (menu Altitude/Slope). As a nice additional feature, GPX information now contains your speed data.

Depth contour lines

If you enter the Download maps menu, you'll notice the file named 'Depth contours ' next to the 'World seamarks' map. This feature gives you access to information about the sea floor, just like the contour lines plugin does with the mountain terrain. To enable it, please go to Configure map - Details.

Driving style setting for cyclists

We all prioritize differently when it comes to reaching your destination. Maybe you are in a hurry and need the fastest route or maybe you'd prefer the safest one. For this case specifically, you can now choose how your route will be built. If you go to the navigation context menu and press the Settings button (the one with the gear wheel), you can select the style you need. Just open the 'Driving style' menu to choose in favor of Speed, Safety or pick the Balanced version.

New map fonts covering more locales

We also added new fonts to the 2.6 version covering more locales. Specifically, we replaced the default ones with Noto Font, which will eliminate the display issues many for many languages. Besides that, the users can download full sets of characters for more convenient map usage. Available for simplified Chinese, traditional Chinese, Japanese, and Korean languages. As always, you can select your map language in Configure map - Map language menu.

Support Open Location Code (OLC)

From now on, OsmAnd supports the Open Location Code. This code represents coordinates of a certain point as a combination of letters and numbers. This way of representation is considered to be easier than the traditional latitude and longitude coordinates. Please read more about the codehere. You can search locations by this code, as well. To try that, please go to Search menu - Address - Coordinates Search and select 'OLC' in Coordinate Format submenu.
