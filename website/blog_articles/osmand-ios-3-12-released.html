OsmAnd 3.12 (iOS)

February 5, 2020

We're glad to announce the new OsmAnd version for iOS.

• Added Route details screen with the altitude and slope profiles, additional data about route surface, steepness, road types and smoothness

In release 3.10 we added the altitude/slope graph.

Now you can click to Details button for get more information about your route. This is feature is in OsmAnd for Android long time. You can put information about route surface, steepness, road types and smoothness in each point of your route.

Сlick to Details of route surface, steepness, road types and smoothness you get a description for each color.

• Added ability to display currently selected graph point on the map

Now you can see all information about each point of your route. For this, you can click to graph or to Details button of Route screen. You can zoom graph and choose any point for viewing needed information.

• Added ability to share a calculated route or save it as a trip

• Added nautical maps with depth contours

• Added the map of Antarctica

• Added ability to Edit online map sources and add them via a special link

• Fixed map rendering issue that caused freezes on latest devices

• Fixed the algorithm for moving time calculation at the recorded Trips screen

• Fixed the crash with automatic track recording during navigation

And remember that only together we can achieve the best results!

New features are coming SOON!
