OsmAnd Ski maps

June 19, 2015

OsmAnd Ski Maps is a plugin allowing you to navigate at winter sports locations. It provides you with a full map of ski pistes, ski lift markers and even colored lines indicating the complexity of each track.

Download the plugin, then use the menu item ‘﻿Configure Map’ to switch the style on.

Ski tracks

Using the plugin, you can easily see the location of the ski pistes, also, you they’ll be marked with different colors showing the complexity of every track. also, you’ll easily find the location of aerialway stations.
